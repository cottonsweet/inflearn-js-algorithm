```
자바스크립트에서는 + - * / % 연산자를 사용할 수 있다.

+ 더하기
- 빼기
* 곱하기
/ 나누기
% 나머지
```

```js
// 더하기
10 + 2 === 12; // true
"js" + "Algorithm" === "jsAlgorithm"; // true
"string" + 123 === "string123"; // true

// 나누기
10 / 2; // 5
10 / 3 = 3.33333333

// 곱셈
10 * 5 // 50
10 * 2 === 20 // true

// 나누기
10 / 2 // 5


// 나머지
10 % 2 // 0
5 % 2 // 1
```
